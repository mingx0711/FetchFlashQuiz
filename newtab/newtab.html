When you click this button, the extension will:
- Scan your vocab list for entries missing etymology, gender, or inflection.
- Fetch any missing data from Wiktionary in the background, pausing 60~300 seconds between each request to avoid overloading the server.
- Not all the vocabs in your vocab list has wiktionary entries, and some might not have gender or eytmology.
For this feature to work, your collection has to be downloaded from this extensions or your collection name must exactly match the language name with an initial capital letter (for example, 'German'), or the json file you upload the vocabs with must have "language" field.
Existing informations of the vocabs will not be overriden.
